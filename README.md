# 📚 Student Management System

A simple **Student Management System** built using **Python** and **Dictionary**. This project allows users to manage student records through a menu-driven interface.

## ✨ Features

* ➕ Add Student
* 📋 View All Students
* 🔍 Search Student by Name
* 🗑️ Delete Student
* ✏️ Update Student Marks
* ✅ Input Validation
* 🚪 Exit Program

## 🛠️ Technologies Used

* Python
* Functions
* Dictionary
* Loops
* Conditional Statements
* User Input
* Exception Handling

## 📸 Menu

```text
📚 ===== STUDENT MANAGEMENT SYSTEM =====
1. Add Student
2. View Students
3. Search Student
4. Delete Student
5. Update Marks
6. Exit
```
